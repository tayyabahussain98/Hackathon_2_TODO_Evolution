# db-conversation

## Description
Prompt for defining SQLModel Conversation and Message models, CRUD operations, and Alembic migration

## Purpose
To guide the implementation of SQLModel-based Conversation and Message models with proper CRUD operations and database migration setup using Alembic.

## Key Elements
- SQLModel Conversation model definition
- SQLModel Message model definition
- Relationship mapping between models
- CRUD operations implementation
- Alembic migration setup
- Database connection configuration
- Data validation and constraints
- Index definitions for performance
