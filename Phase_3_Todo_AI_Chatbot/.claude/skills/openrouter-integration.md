# openrouter-integration

## Description
Prompt for configuring OpenAI Agents SDK with OpenRouter API key, base_url, and headers

## Purpose
To guide the configuration of OpenAI Agents SDK to work with OpenRouter API, including proper API key handling, base URL setup, and header configurations.

## Key Elements
- OpenRouter API key configuration
- Base URL setup for OpenRouter
- Header configuration and authentication
- OpenAI Agents SDK initialization
- API key security best practices
- Environment variable handling
- Error handling for API connections
- Configuration validation
