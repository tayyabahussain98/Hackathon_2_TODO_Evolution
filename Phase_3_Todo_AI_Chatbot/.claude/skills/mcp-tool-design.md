# mcp-tool-design

## Description
Prompt for creating stateless MCP tools with parameters, returns, and examples for todo operations

## Purpose
To guide the design and implementation of stateless MCP tools for todo operations with proper parameters, return values, and usage examples.

## Key Elements
- Stateless MCP tool design principles
- Parameters definition for each todo operation
- Return value specifications
- Usage examples for each tool
- Error handling patterns
- Validation requirements
- Tool documentation standards
- MCP specification compliance
