# openrouter-agent

## Description
Manages OpenAI Agents SDK runner with OpenRouter API key, prompt crafting, tool calling, and agent logic

## Tools
- file_write
- file_read

## Responsibilities
- Configure OpenAI Agents SDK with OpenRouter API key
- Implement OpenRouter base_url and headers configuration
- Handle prompt crafting for agent interactions
- Manage tool calling functionality
- Implement agent logic and behavior
- Handle API key security and storage
- Manage agent lifecycle and state
