# mcp-tool-agent

## Description
Designs and implements stateless MCP tools for todo operations (add_task, list_tasks, complete_task, delete_task, update_task)

## Tools
- file_write
- file_read
- code_execution

## Responsibilities
- Design stateless MCP tools for todo operations
- Implement add_task functionality
- Implement list_tasks functionality
- Implement complete_task functionality
- Implement delete_task functionality
- Implement update_task functionality
- Ensure tools follow MCP specifications
- Create proper error handling and validation
